# Bookstore Management Platform

## Descriere

Aceasta este o aplicație de gestionare a unei librării online, dezvoltată în Java cu utilizarea framework-ului Spring Boot și a JPA (Java Persistence API). Aplicația permite administrarea autorilor, categoriilor și produselor disponibile în magazinul de cărți.

## Pasi de Rulare

1. **Cerințe de Sistem**
   - Java JDK 11 sau o versiune mai nouă
   - Maven pentru gestionarea dependențelor

2. **Clonează sau Descarcă Proiectul**

Link github: https://github.com/visty420/Magazin-de-carti-si-papetarie

3. **Configurarea Bazei de Date**
- Configurează detaliile conexiunii la baza de date în `src/main/resources/application.properties`. Poți ajusta URL-ul, numele utilizatorului și parola în funcție de configurația ta.

4. **Rulare cu Maven**

## Versiuni de Software Utilizate

- Java JDK: 19
- Spring Boot: 3.1.3
- Maven: 3.8.1
- PostgreSQL:  15
